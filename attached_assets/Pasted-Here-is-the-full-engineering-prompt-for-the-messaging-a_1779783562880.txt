## What We Are Building

Two distinct communication systems that work together:

**System 1 — Club Chat (Group Channel)**
Every club has one group chat. All club members (admin, coaches, analysts, scouts, athletes) are in it. Messages appear for everyone in real time exactly like a WhatsApp group.

**System 2 — Direct Messages (1-on-1)**
Any user can message any other user directly. Conversations are private, threaded and organised exactly like WhatsApp — a list of conversations, tap one to open the thread.

Both systems live under a **Messages** tab in every dashboard. Same UI pattern for every role. Simple, fast, real time.

---

## Architecture Overview

```
Firestore Collections:

conversations/{conversationId}
  → type: "group" | "direct"
  → participants: [userId1, userId2, ...]
  → clubId (for group chats)
  → lastMessage: { body, senderId, sentAt }
  → lastMessageAt (for sorting)
  → createdAt

conversations/{conversationId}/messages/{messageId}
  → senderId
  → senderName
  → senderRole
  → senderAvatar
  → body
  → sentAt: serverTimestamp()
  → readBy: { userId: timestamp, ... }
  → isDeleted: false

users/{userId}
  → conversationIds: [] (array of conversationIds user belongs to)
```

## Part 1 — Club Group Chat

### How It Works
- One group conversation document per club
- Created automatically when a club is registered
- All club members auto-added to participants array when they join
- All club members auto-removed when they leave
- Messages appear in real time for all participants

### Step 1 — Create Club Group Chat on Club Registration

```javascript
// Called once when club is first created
async function createClubGroupChat(clubId, clubName, adminId) {
  const conversationRef = doc(db, 'conversations', `club_${clubId}`)
  
  await setDoc(conversationRef, {
    type: 'group',
    clubId,
    name: clubName,
    avatar: null, // club badge URL
    participants: [adminId],
    participantRoles: { [adminId]: 'club_admin' },
    lastMessage: null,
    lastMessageAt: serverTimestamp(),
    createdAt: serverTimestamp()
  })

  // Add conversation to admin's list
  await updateDoc(doc(db, 'users', adminId), {
    conversationIds: arrayUnion(`club_${clubId}`)
  })
}
```

### Step 2 — Auto-Add Member to Club Chat When They Join

```javascript
// Call this every time ANY user joins a club
// (athlete joins squad, coach linked to club,
//  analyst linked, scout linked)
async function addMemberToClubChat(clubId, userId, userRole) {
  const chatRef = doc(db, 'conversations', `club_${clubId}`)
  
  await updateDoc(chatRef, {
    participants: arrayUnion(userId),

  // Add to user's conversation list
  await updateDoc(doc(db, 'users', userId), {
    conversationIds: arrayUnion(`club_${clubId}`)
  })

// Call this when user leaves or is removed from club
async function removeMemberFromClubChat(clubId, userId) {
  const chatRef = doc(db, 'conversations', `club_${clubId}`)
  
  await updateDoc(chatRef, {
    participants: arrayRemove(userId)
  })

  await updateDoc(doc(db, 'users', userId), {
    conversationIds: arrayRemove(`club_${clubId}`)
  })
}
```

### Step 3 — Send a Message to the Group

```javascript
async function sendGroupMessage(conversationId, sender, body) {
  const msgRef = doc(collection(db, 'conversations', conversationId, 'messages'))
  
  const message = {
    senderId: sender.uid,
    senderName: sender.name,
    senderRole: sender.role,
    senderAvatar: sender.avatarUrl,
    body: body.trim(),
    sentAt: serverTimestamp(),
    readBy: { [sender.uid]: new Date() },
    isDeleted: false
  }

  // Write message + update conversation last message atomically
  const batch = writeBatch(db)
  
  batch.set(msgRef, message)
  batch.update(doc(db, 'conversations', conversationId), {
    lastMessage: {
      body: body.trim(),
      senderId: sender.uid,
      senderName: sender.name
    },
    lastMessageAt: serverTimestamp()
  })

  await batch.commit()
}
```

### Step 4 — Real-Time Message Listener

```javascript
function subscribeToMessages(conversationId, onMessages) {
  const q = query(
    collection(db, 'conversations', conversationId, 'messages'),
    orderBy('sentAt', 'asc'),
    limitToLast(100)
  )

  return onSnapshot(q, (snap) => {
    const messages = snap.docs.map(d => ({
      id: d.id,
      ...d.data()
    }))
    onMessages(messages)
  })
}
// Returns unsubscribe function — call on component unmount
```

### Step 5 — Mark Messages as Read

```javascript
// Call when user opens the conversation
async function markMessagesRead(conversationId, userId) {
  const q = query(
    collection(db, 'conversations', conversationId, 'messages'),
    where(`readBy.${userId}`, '==', null)
  )
  
  const unread = await getDocs(q)
  const batch = writeBatch(db)
  
  unread.docs.forEach(d => {
    batch.update(d.ref, {
      [`readBy.${userId}`]: serverTimestamp()
    })
  })
  
  await batch.commit()
}
```

## Part 2 — Direct Messages (1-on-1)

### How It Works
- Any user can start a DM with any other user they have a relationship with
- Conversation document is shared between both users
- Thread is private — only the two participants can read it
- Conversation list shows all DMs sorted by most recent message

### Step 1 — Start or Open a Direct Message

```javascript
// Creates conversation if it doesn't exist
// Returns existing conversation if it does
async function getOrCreateDirectMessage(userId1, userId2) {
  // Deterministic ID — always the same for any two users
  const ids = [userId1, userId2].sort()
  const conversationId = `dm_${ids[0]}_${ids[1]}`
  
  const convRef = doc(db, 'conversations', conversationId)
  const existing = await getDoc(convRef)
  
  if (existing.exists()) {
    return conversationId
  }

  // Fetch both user profiles
  const [user1Doc, user2Doc] = await Promise.all([
    getDoc(doc(db, 'users', userId1)),
    getDoc(doc(db, 'users', userId2))
  ])
  
  const user1 = user1Doc.data()
  const user2 = user2Doc.data()

  await setDoc(convRef, {
    type: 'direct',
    participants: [userId1, userId2],
    participantProfiles: {
      [userId1]: {
        name: user1.name,
        avatar: user1.avatarUrl,
        role: user1.role
      },
      [userId2]: {
        name: user2.name,
        avatar: user2.avatarUrl,
        role: user2.role
      }
    },
    lastMessage: null,
    lastMessageAt: serverTimestamp(),
    createdAt: serverTimestamp()
  })

  // Add to both users' conversation lists
  await Promise.all([
    updateDoc(doc(db, 'users', userId1), {
      conversationIds: arrayUnion(conversationId)
    }),
    updateDoc(doc(db, 'users', userId2), {
      conversationIds: arrayUnion(conversationId)
    })
  ])

  return conversationId
}
```

### Step 3 — Load All Conversations for a User

```javascript
async function getUserConversations(userId) {
  const userDoc = await getDoc(doc(db, 'users', userId))
  const conversationIds = userDoc.data().conversationIds || []
  
  if (conversationIds.length === 0) return []

  // Fetch all conversations in parallel
  const convDocs = await Promise.all(
    conversationIds.map(id => getDoc(doc(db, 'conversations', id)))
  )

  return convDocs
    .filter(d => d.exists())
    .map(d => ({ id: d.id, ...d.data() }))
    .sort((a, b) => {
      // Sort by last message time descending
      const ta = a.lastMessageAt?.toMillis() || 0
      const tb = b.lastMessageAt?.toMillis() || 0
      return tb - ta
    })
}
```

exports.updateUnreadCounts = functions.firestore
  .document('conversations/{convId}/messages/{msgId}')
  .onCreate(async (snap, ctx) => {
    const message = snap.data()

    await batch.commit()

## Part 4 — Firestore Security Rules

```javascript
// firestore.rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Conversations — only participants can read/write
    match /conversations/{convId} {
      allow read: if request.auth.uid in resource.data.participants;
      allow create: if request.auth.uid in request.resource.data.participants;
      allow update: if request.auth.uid in resource.data.participants;
      
      // Messages — only participants can read/write
      match /messages/{msgId} {
        allow read: if request.auth.uid in 
          get(/databases/$(database)/documents/conversations/$(convId))
            .data.participants;
        allow create: if request.auth.uid in 
          get(/databases/$(database)/documents/conversations/$(convId))
            .data.participants
          && request.resource.data.senderId == request.auth.uid;
        allow update: if request.auth.uid in
          get(/databases/$(database)/documents/conversations/$(convId))
            .data.participants;
      }

      // Unread counts — participant can only write their own
      match /unreadCounts/{userId} {
        allow read: if request.auth.uid == userId;
        allow write: if false; // Cloud Function only
      }
    }
  }
}
```

function ConversationItem({ conversation, currentUser, onPress }) {
  const isGroup = conversation.type === 'group'
  
  // For DMs: show the OTHER person's name
  const otherUserId = conversation.participants
    .find(id => id !== currentUser.uid)
  const otherProfile = conversation.participantProfiles?.[otherUserId]
  
  const displayName = isGroup
    ? conversation.name
    : otherProfile?.name || 'Unknown'
  
  const displayAvatar = isGroup
    ? conversation.avatar
    : otherProfile?.avatar
  
  const lastMsg = conversation.lastMessage
  const preview = lastMsg
    ? lastMsg.senderId === currentUser.uid
      ? `You: ${lastMsg.body}`
      : lastMsg.body
    : 'No messages yet'

  return (
    <div onClick={onPress} className="conversation-item">
      <Avatar src={displayAvatar} name={displayName} isGroup={isGroup} />
      <div className="conversation-info">
        <div className="conversation-header">
          <span className="name">{displayName}</span>
          <span className="time">
            {formatTime(conversation.lastMessageAt)}
          </span>
        </div>
        <div className="conversation-preview">
          <span className="preview-text">{preview}</span>
          {unreadCount > 0 && (
            <span className="unread-badge">{unreadCount}</span>
          )}
        </div>
      </div>
    </div>
  )
}

### Chat Thread Component

```jsx
function ChatThread({ conversationId, currentUser }) {
  const [messages, setMessages] = useState([])
  const [input, setInput] = useState('')
  const bottomRef = useRef(null)

  // Subscribe to real-time messages
  useEffect(() => {
    const unsub = subscribeToMessages(conversationId, (msgs) => {
      setMessages(msgs)
      // Auto-scroll to bottom on new message
      setTimeout(() => bottomRef.current?.scrollIntoView({ behavior: 'smooth' }), 50)
    })
    
    // Mark as read when opened
    markMessagesRead(conversationId, currentUser.uid)
    
    return unsub // Cleanup on unmount
  }, [conversationId])

  async function handleSend() {
    if (!input.trim()) return
    const body = input.trim()
    setInput('') // Clear immediately for responsiveness
    await sendGroupMessage(conversationId, currentUser, body)
  }

  function handleKeyDown(e) {
    if (e.key === 'Enter' && !e.shiftKey) {
      e.preventDefault()
      handleSend()
    }
  }

  return (
    <div className="chat-thread">
      {/* Message list */}
      <div className="messages-scroll">
        {messages.map((msg, i) => (
          <MessageBubble
            key={msg.id}
            message={msg}
            isOwn={msg.senderId === currentUser.uid}
            showSender={conversation.type === 'group'}
            showAvatar={
              i === 0 ||
              messages[i-1].senderId !== msg.senderId
            }
          />
        ))}
        <div ref={bottomRef} />
      </div>

      {/* Input bar */}
      <div className="input-bar">
        <textarea
          value={input}
          onChange={e => setInput(e.target.value)}
          onKeyDown={handleKeyDown}
          placeholder="Message..."
          rows={1}
        />
        <button onClick={handleSend} disabled={!input.trim()}>
          Send
        </button>
      </div>
    </div>
  )
}
```

function MessageBubble({ message, isOwn, showSender, showAvatar }) {
  return (
    <div className={`bubble-wrapper ${isOwn ? 'own' : 'other'}`}>
      {/* Show avatar for group chats, not own messages */}
      {!isOwn && showAvatar && (
        <Avatar src={message.senderAvatar} name={message.senderName} size={28} />

          {isOwn && <ReadReceipt message={message} participants={participants} />}

### Read Receipts Component

```jsx
function ReadReceipt({ message, participants }) {
  const readCount = Object.keys(message.readBy || {}).length - 1 // exclude sender
  const totalRecipients = participants.length - 1
  
  if (readCount === 0) return <span className="receipt">✓</span>          // Sent
  if (readCount < totalRecipients) return <span className="receipt">✓✓</span>  // Delivered
  return <span className="receipt read">✓✓</span>                          // Read (blue)
}
```

## Part 7 — Who Can Message Whom

```
CLUB ADMIN
  → Group chat: all club members
  → DM: any coach, analyst, scout or athlete in their club

COACH
  → Group chat: all club members
  → DM: club admin, fellow coaches, analysts, any athlete in squad

ANALYST
  → Group chat: all club members
  → DM: club admin, head coach, fellow analysts

SCOUT
  → Group chat: all club members (if affiliated with club)
  → DM: athletes (after scout request accepted), club admin

ATHLETE
  → Group chat: all club members (if in a club)
  → DM: coach, club admin, scouts who they accepted requests from,
         fellow athletes in same squad
```

```javascript
// searchRelatedUsers — returns users this person can DM
async function searchRelatedUsers(currentUser, query) {
  const role = currentUser.role
  let userIds = []

  if (role === 'club_admin') {
    // All members of their club
    const club = await getDoc(doc(db, 'clubs', currentUser.clubId))
    userIds = club.data().members || []
  }

  if (role === 'coach') {
    // Squad athletes + club admin + fellow coaches
    const squadAthletes = await getSquadAthleteIds(currentUser.uid)
    userIds = [...squadAthletes, currentUser.clubAdminId]
  }

  if (role === 'scout') {
    // Athletes with accepted scout requests
    const accepted = await getDocs(query(
      collection(db, 'scoutRequests'),
      where('scoutId', '==', currentUser.uid),
      where('status', '==', 'accepted')
    ))
    userIds = accepted.docs.map(d => d.data().athleteId)
  }

  if (role === 'athlete') {
    // Coach + club admin + accepted scouts + squad teammates
    userIds = [
      currentUser.coachId,
      currentUser.clubAdminId,
      ...await getAcceptedScoutIds(currentUser.uid),
      ...await getSquadTeammateIds(currentUser.uid)
    ].filter(Boolean)
  }

  // Filter by search query
  const users = await Promise.all(userIds.map(id => getDoc(doc(db, 'users', id))))
  return users
    .filter(d => d.exists())
    .map(d => ({ uid: d.id, ...d.data() }))
    .filter(u => u.name.toLowerCase().includes(query.toLowerCase()))
}
```

## Part 8 — Notifications for New Messages

```javascript
// Cloud Function: notify user of new message
exports.onNewMessage = functions.firestore
  .document('conversations/{convId}/messages/{msgId}')
  .onCreate(async (snap, ctx) => {
    const message = snap.data()
    const { convId } = ctx.params
    
    const convDoc = await getDoc(doc(db, 'conversations', convId))
    const conv = convDoc.data()
    
    // Notify all participants except sender
    const recipients = conv.participants
      .filter(uid => uid !== message.senderId)
    
    // In-app notifications
    const batch = writeBatch(db)
    recipients.forEach(uid => {
      const notifRef = doc(collection(db, `notifications/${uid}/items`))
      batch.set(notifRef, {
        type: 'new_message',
        conversationId: convId,
        conversationType: conv.type,
        senderName: message.senderName,
        preview: message.body.substring(0, 60),
        read: false,
        createdAt: serverTimestamp()
      })
    })
    await batch.commit()

    // SMS for DMs only (not group chat — too noisy)
    if (conv.type === 'direct') {
      for (const uid of recipients) {
        const userDoc = await getDoc(doc(db, 'users', uid))
        const phone = userDoc.data().phone
        if (phone) {
          await sendSMS(phone,
            `TalentGraph: New message from ${message.senderName}. ` +
            `"${message.body.substring(0, 50)}..." ` +
            `Reply at talentgraph.co.ke/messages`
          )
        }
      }
    }
  })
```

## Part 10 — Messages Tab Integration (Per Dashboard)

Add this to every dashboard sidebar:

```jsx
// In every dashboard layout
function MessagesTab({ currentUser }) {
  const [conversations, setConversations] = useState([])
  const [activeConv, setActiveConv] = useState(null)
  const [showNewDM, setShowNewDM] = useState(false)

  useEffect(() => {
    getUserConversations(currentUser.uid).then(setConversations)
    
    // Real-time updates to conversation list
    const unsub = onSnapshot(
      query(
        collection(db, 'conversations'),
        where('participants', 'array-contains', currentUser.uid),
        orderBy('lastMessageAt', 'desc')
      ),
      snap => setConversations(snap.docs.map(d => ({ id: d.id, ...d.data() })))
    )
    return unsub
  }, [])

  // Separate groups from DMs
  const groups = conversations.filter(c => c.type === 'group')
  const directs = conversations.filter(c => c.type === 'direct')

  if (activeConv) {
    return (
      <ChatThread
        conversationId={activeConv.id}
        conversation={activeConv}
        currentUser={currentUser}
        onBack={() => setActiveConv(null)}
      />
    )
  }

  return (
    <div className="messages-tab">
      <div className="messages-header">
        <h2>Messages</h2>
        <button onClick={() => setShowNewDM(true)}>✏️</button>
      </div>

      <input className="search-bar" placeholder="Search conversations..." />

      {groups.length > 0 && (
        <>
          <div className="section-label">GROUPS</div>
          {groups.map(conv => (
            <ConversationItem
              key={conv.id}
              conversation={conv}
              currentUser={currentUser}
              onPress={() => setActiveConv(conv)}
            />
          ))}
        </>
      )}

      {directs.length > 0 && (
        <>
          <div className="section-label">DIRECT MESSAGES</div>
          {directs.map(conv => (
            <ConversationItem
              key={conv.id}
              conversation={conv}
              currentUser={currentUser}
              onPress={() => setActiveConv(conv)}
            />
          ))}
        </>
      )}

      {showNewDM && (
        <NewDirectMessage
          currentUser={currentUser}
          onConversationCreated={(id) => {
            setShowNewDM(false)
            setActiveConv({ id })
          }}
        />
      )}
    </div>
  )
}
```
